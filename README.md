# CSCE-314
All projects in CSCE 314. Do not copy code.

## Cloning
To clone this repository, make sure **git** is installed on your system. This can be done using **pip** with the following command:
```bash
pip install git
```
Next, clone the repository using the https link:
```bash
git clone <https link>
```

## Usage
Most of the code for this project is designed to run in Eclipse. The GUI part of the project was developed on WindowBuilder. Make sure that the WindowBuilder package is installed for your Eclipse version.
